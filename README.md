# 📘 Thesis Begeleider Tool - Erasmus Universiteit

**Senior thesisbegeleider voor masterstudenten**  
Inhoud, methodologie, lay-out, bronvermelding (APA) en AI-detectie.

## 🎯 Functionaliteiten

| Check | Beschrijving |
|-------|---------------|
| ✅ Titelblad | Controleert op naam, studentnummer, titel, opleiding, datum, begeleider |
| ✅ Methodologie | Toetst design, steekproef, meetinstrument, procedure en analyse |
| ✅ Bronvermelding | Verifieert APA 7e editie (journals, auteurs, & en et al.) |
| ✅ AI-detectie | Herkent stylometrische patronen die wijzen op AI-gebruik |
| ✅ Stappenplan | Genereert concrete acties om de thesis te laten slagen |
