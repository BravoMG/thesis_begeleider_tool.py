# APA7 NL - Installatiegids voor Microsoft Word

Deze gids helpt je bij het installeren van de APA7 NL-stijl in Microsoft Word, zodat je automatisch correcte APA 7e editie verwijzingen kunt genereren.

## 📥 Download

| Besturingssysteem | Downloadlink |
|------------------|--------------|
| **Windows** | [APASeventhEditionNL-1.0.2_1.zip](https://auteursrechten.nl/wp-content/uploads/2025/10/APASeventhEditionNL-1.0.2_1.zip) |
| **MacOS** | [APASeventhEditionNL-1.0.2-1.xsl](https://auteursrechten.nl/wp-content/uploads/2025/06/APASeventhEditionNL-1.0.2-1.xsl) |

---

## 🪟 Windows Installatie

1. Download het zipbestand
2. Ga naar de map **Downloads**
3. Klik met de **rechtermuisknop** op het zip-bestand
4. Kies **"Alles uitpakken"**
5. Er opent zich een nieuwe map
6. Klik op **"Installeer APASeventhEditionNL voor Word"**
7. Klik op **"Uitvoeren"**
8. **Start Word opnieuw op** (belangrijk!)

---

## 🍎 MacOS Installatie

1. Download het `.xsl`-bestand
2. **Kopieer** het bestand (⌘ C)
3. Ga naar de map: `/Applications/Microsoft Word.app/Contents/Resources/style`
   - Gebruik **⌘ ⇧ G** en plak de mapnaam
4. **Plak** het bestand in deze map (gebruik **⌘ V**)
   - ⚠️ **Let op:** Niet slepen, want dan kan de kopie mislukken
5. Start Word opnieuw op

---

## ✅ Controleren of het werkt

1. Open Word
2. Ga naar het tabblad **"Verwijzingen"**
3. Klik op de pijl bij **"Stijl"**
4. Je zou **"APA7 NL"** in de lijst moeten zien

![APA7 NL in Word](https://auteursrechten.nl/wp-content/uploads/2025/06/APA7-NL-in-Word.png)

## 📝 Hoe gebruik je APA7 NL?

### Een verwijzing toevoegen

1. Ga naar de plek in de tekst waar de verwijzing moet komen
2. Klik op **"Citaat invoegen"** → **"Nieuwe bron invoegen"**
3. Vul de gegevens van de bron in
4. Klik op **OK**
